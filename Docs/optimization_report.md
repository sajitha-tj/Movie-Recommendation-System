# Optimization Report

## Introduction

This report outlines the optimizations implemented in the Movie Recommendation System, focusing on improving the performance of hotspots identified in the program using profiling tools ([`perf`](https://perfwiki.github.io/main/)). The optimizations were aimed at reducing the execution time and enhancing the efficiency of the algorithms used.

## Methodology

### Initial Code Review and Fixes

The intial [Movie-Recommendation-System](https://github.com/shubham-bhatnagar/Movie-Recommendation-System) program contained some legacy code and problems that were identified and fixed to ensure the program runs correctly. The fixed code can be found in the [fixed](https://github.com/sajitha-tj/Movie-Recommendation-System/tree/fixed) branch.

### Baseline Performance Measurement

Initial program was heavily dependent on user input, which was not ideal for performance testing. To address this, the program was modified to run with predefined inputs, allowing for consistent performance measurements. Modified code can be found in the [baseline](https://github.com/sajitha-tj/Movie-Recommendation-System/tree/baseline) branch. This baseline version was then executed for 5 consecutive runs and following are the time taken for recommendation in each run:

| Run | Time (seconds) |
|-----|----------------|
| 1   | 0.17           |
| 2   | 0.14           |
| 3   | 0.14           |
| 4   | 0.15           |
| 5   | 0.16           |

The average time taken for recommendation in the baseline version was calculated to be approximately 0.152 seconds.
